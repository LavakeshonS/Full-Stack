CheatSheet - HTML TABLE elements
TABLE Elements
Element Name Description
<table> Table A table in HTML consists of table cells inside rows and columns.
<thead> Table header The headings of table columns encapsulated in table rows.
<tbody> Table body The semantic element that will contain all table data other than table heading and table footer content.
<tfoot> Table footer The element summarizes content at the end of a table.
<th> Table heading Defines the headings of table columns encapsulated in table rows.
<tr> Table row It's used to add rows to a table before adding table data and table headings.
<td> Table data The table data element, can be nested inside a table row element, to add a cell of data to a table.
TABLE Attributes
Element Name Description
<rowspan> Row Span The element indicates how many rows that particular cell should span within the table.
<colspan> Column Span The element indicates how many columns that particular cell should span within the table.
